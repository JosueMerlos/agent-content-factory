# OpenCode

Usa este agente cuando la tarea necesite:

- flujos reproducibles,
- colaboración técnica,
- integración con tooling abierto,
- adaptación a stacks mixtos.

## Cómo consumir skills

OpenCode debe apuntar a:

`skills/<dominio>/<skill>/adapters/opencode/SKILL.md`

## Buenas prácticas

- Evita depender de capacidades propietarias no garantizadas.
- Define formatos de salida simples y portables.
- Documenta cualquier diferencia experimental frente a Claude o Codex.
