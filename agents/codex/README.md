# Codex

Usa este agente cuando la tarea necesite:

- ejecución estructurada,
- automatización,
- consistencia operativa,
- generación de artefactos accionables.

## Cómo consumir skills

Codex debe apuntar a:

`skills/<dominio>/<skill>/adapters/codex/SKILL.md`

## Buenas prácticas

- Mantén instrucciones concretas y verificables.
- Explicita formato de salida y criterios de aceptación.
- Usa ejemplos cercanos a flujos reales del pipeline.
