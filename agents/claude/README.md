# Claude

Usa este agente cuando la tarea necesite:

- redacción extensa,
- refinamiento de tono,
- síntesis narrativa,
- iteración editorial.

## Cómo consumir skills

Claude debe apuntar a los adaptadores en:

`skills/<dominio>/<skill>/adapters/claude/SKILL.md`

## Buenas prácticas

- Prioriza contexto editorial y audiencia.
- Pide entregables largos solo cuando de verdad agreguen valor.
- Mantén consistencia con `skill.yaml` para no desalinear inputs y outputs.
