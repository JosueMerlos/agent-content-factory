# Preset: YouTube Longform

## Use When

- el usuario pide video largo para YouTube,
- busca profundidad, autoridad o conversion,
- la duracion esperada supera 3 minutos.

## Default Assumptions

- estructura con retencion por segmentos,
- CTA mas desarrollado,
- posibilidad de reutilizacion a shorts,
- mayor necesidad de thumbnail y empaquetado final.

## Recommended Skills

1. `topic-research`
2. `script-writer`
3. `voiceover-generator`
4. `text-to-video`
5. `viral-clips-generator`
6. `thumbnail-generator`
7. `copyright-compliance`
8. `seo-and-publishing`

## Expected Deliverables

- angulo,
- guion completo,
- narracion,
- plan visual,
- clips para shorts,
- thumbnail,
- package de publicacion.
