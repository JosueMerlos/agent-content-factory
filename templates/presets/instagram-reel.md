# Preset: Instagram Reel

## Use When

- el usuario pide un Reel,
- el contenido debe verse bien en Instagram,
- el objetivo es awareness, comunidad o autoridad ligera.

## Default Assumptions

- formato vertical,
- tono limpio y rapido,
- visuales legibles en movil,
- CTA discreto o conversacional.

## Recommended Skills

1. `topic-research`
2. `script-writer`
3. `text-to-video`
4. `copyright-compliance`

## Expected Deliverables

- concepto,
- guion corto,
- plan visual,
- copy/caption,
- revision de riesgo.
