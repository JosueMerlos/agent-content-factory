# Code of Conduct

## Our Standard

Queremos que este proyecto sea un espacio útil, respetuoso y colaborativo.

Se espera que quienes participen:

- actúen con respeto y buena fe,
- den feedback claro sin descalificar,
- eviten lenguaje hostil, discriminatorio o intimidante,
- prioricen colaboración y aprendizaje por encima de ego o fricción innecesaria.

## Unacceptable Behavior

No se tolera:

- acoso o ataques personales,
- lenguaje discriminatorio o degradante,
- doxxing o exposición de información privada,
- conducta agresiva que haga difícil colaborar de forma segura.

## Scope

Este código de conducta aplica a:

- issues,
- pull requests,
- discusiones del repositorio,
- cualquier interacción razonablemente asociada a este proyecto.

## Enforcement

Los mantenedores pueden moderar, editar, ocultar o remover contribuciones que rompan estas expectativas.

## Reporting

Si observas un problema de conducta, repórtalo por el canal que definan los mantenedores del repositorio.

## Note

Este archivo es una base ligera para lanzar el repositorio. Si el proyecto crece, conviene reemplazarlo por una política más formal, por ejemplo una variante completa de Contributor Covenant.
