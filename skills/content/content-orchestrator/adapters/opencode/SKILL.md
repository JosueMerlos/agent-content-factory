# Content Orchestrator for OpenCode

Usa esta skill para enrutar pedidos naturales hacia un workflow reproducible.

## Instrucciones

1. Separa interpretación, preset, workflow y entrega.
2. Usa nombres de preset estables.
3. Mantén la salida portable y fácil de automatizar.

## Formato sugerido

- `request_interpretation`
- `selected_preset`
- `workflow_plan`
- `user_facing_deliverables`
- `next_action`
