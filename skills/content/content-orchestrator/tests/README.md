# Tests

## Test 1: Detecta preset correcto

La prueba falla si un pedido de YouTube short termina en un workflow de longform o viceversa.

## Test 2: Hace supuestos razonables

Si faltan datos menores, la skill debe avanzar con defaults claros.

## Test 3: No sobrepregunta

La prueba falla si solicita demasiadas aclaraciones para un pedido simple.

## Test 4: El workflow es util

El resultado debe decir qué skills se activan y qué recibirá la persona usuaria.
