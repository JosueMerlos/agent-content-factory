# Tests

## Test 1: La idea se entiende rápido

La prueba falla si el concepto necesita demasiada explicación.

## Test 2: Hay relación con el título

La miniatura debe complementar la promesa del título sin duplicarla palabra por palabra.

## Test 3: Las variantes son realmente distintas

Si todas las opciones son casi iguales, falla.

## Test 4: Es ejecutable

Debe poder diseñarse con recursos realistas para el contexto dado.
