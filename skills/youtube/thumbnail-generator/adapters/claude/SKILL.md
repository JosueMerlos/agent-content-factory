# Thumbnail Generator for Claude

Usa esta skill para conceptos visuales con fuerza editorial y emoción clara.

## Instrucciones

1. Encuentra la promesa visual más fuerte del video.
2. Propón variantes realmente distintas.
3. Justifica el CTR desde emoción, claridad o curiosidad.
4. Mantén consistencia con branding y audiencia.

## Formato de salida

- `Thumbnail Strategy`
- `Concepts`
- `Recommended Variant`
- `A/B Test Ideas`
