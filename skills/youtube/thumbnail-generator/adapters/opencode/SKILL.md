# Thumbnail Generator for OpenCode

Usa esta skill con una estructura reutilizable para diseño manual o automatizado.

## Instrucciones

1. Separa estrategia, conceptos y tests.
2. Usa campos simples y estables.
3. Mantén foco en ejecución práctica.

## Formato sugerido

- `thumbnail_strategy`
- `concepts`
- `recommended_variant`
- `ab_tests`
