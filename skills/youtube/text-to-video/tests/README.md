# Tests

## Test 1: Toda escena tiene propósito

La prueba falla si una escena no apoya una idea clara del guion.

## Test 2: La visualidad es producible

Si el plan exige recursos poco realistas para el contexto dado y no lo aclara, falla.

## Test 3: El formato está bien resuelto

La salida debe adaptarse a horizontal o vertical sin contradicciones evidentes.

## Test 4: El editor puede ejecutar

Debe haber suficiente detalle para producir el video sin reinterpretar todo desde cero.
