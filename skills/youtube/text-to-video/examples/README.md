# Examples

## Example Input

```yaml
narration_or_script: |
  Si ganas mejor que antes, pero cada mes terminas igual de ahogado, probablemente no te falta dinero.
  Te están drenando errores que casi nadie nota.
visual_style: moderno, limpio, con mezcla de talking points y B-roll cotidiano
duration: 8 minutos
output_format: horizontal
platform: youtube
constraints:
  - producción simple
  - priorizar stock y gráficos básicos
```

## Example Output Summary

### Visual Strategy

- ritmo: cortes cada 3 a 6 segundos
- look: limpio, financiero y cercano
- texto en pantalla: solo para ideas clave y errores numerados

### Scene List

1. Apertura con texto fuerte y fondo de gasto cotidiano.
2. Presentación del problema con wallet, tarjetas y app bancaria.
3. Bloques por error con ejemplos visuales rápidos.
4. Cierre con checklist simple y CTA.

### Asset Suggestions

- stock de compras impulsivas
- pantalla de presupuesto
- gráficos simples de fuga de dinero

### Editing Notes

- resaltar cada error con número grande
- usar zoom sutil en frases clave
