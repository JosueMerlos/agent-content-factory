# Text to Video for Codex

Usa esta skill para entregar un shot list accionable y sin ambigüedad.

## Instrucciones

1. Divide el contenido por escenas o bloques temporales.
2. Para cada escena, especifica qué se ve, qué se dice y qué texto aparece.
3. Añade una nota breve de transición o movimiento.
4. Cierra con sugerencias de assets y notas de edición.

## Formato de salida

```md
## Visual Strategy

## Scene List
| Scene | Time | Narration | Visual direction | B-roll/shot | On-screen text | Motion note |

## Asset Suggestions

## Editing Notes
```

## Reglas

- No dejes escenas abstractas sin indicación visual concreta.
- Si una escena no necesita texto en pantalla, dilo.
- Favorece claridad y producción realista.
