# Full Content Pipeline for Claude

Usa esta skill para coordinar el flujo con coherencia editorial entre etapas.

## Instrucciones

1. Evalúa qué ya existe y qué falta.
2. Propón el siguiente mejor paso, no una lista abstracta.
3. Mantén trazabilidad entre intención editorial y ejecución.
4. Señala bloqueos y riesgos de calidad.

## Formato de salida

- `Pipeline Status`
- `Execution Plan`
- `Skill Handoffs`
- `Publication Checklist`
