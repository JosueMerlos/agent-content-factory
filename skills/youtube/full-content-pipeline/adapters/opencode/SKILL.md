# Full Content Pipeline for OpenCode

Usa esta skill para modelar el pipeline como un flujo portable y automatizable.

## Instrucciones

1. Separa estado, plan, handoffs y checklist.
2. Usa etiquetas consistentes para cada etapa.
3. Mantén el resultado fácil de integrar con tooling externo.

## Formato sugerido

- `pipeline_status`
- `execution_plan`
- `skill_handoffs`
- `publication_checklist`
