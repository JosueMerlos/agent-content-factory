# Tests

## Test 1: Suena hablado

La prueba falla si el texto final se siente como artículo escrito en vez de narración oral.

## Test 2: Tiene control de ritmo

Debe haber pausas, énfasis o notas de performance suficientes para guiar una locución.

## Test 3: Respeta la duración

Si el texto queda claramente fuera de la duración objetivo sin avisarlo, la prueba falla.

## Test 4: Handoff usable

La salida debe poder alimentar `text-to-video` sin perder estructura ni intención.
