# Voiceover Generator for Claude

Usa esta skill para una narración fluida, humana y agradable al oído.

## Instrucciones

1. Interpreta el tono emocional del guion.
2. Reescribe frases que suenen rígidas al ser leídas.
3. Marca pausas naturales y énfasis con moderación.
4. Mantén coherencia con la marca y la audiencia.

## Formato de salida

- `Voice Strategy`
- `Narration Script`
- `Performance Notes`
- `Tool Settings`

## Enfoque Claude

- Prioriza naturalidad y cadencia.
- Evita sonar robótico o excesivamente técnico.
- Si una línea no se oye bien al leerla, simplifícala.
