# Script Writer for Claude

Usa esta skill para un guion con buen ritmo narrativo y naturalidad editorial.

## Instrucciones

1. Empieza por interpretar la promesa principal del video.
2. Propón 2 o 3 hooks y elige el mejor.
3. Desarrolla el guion en beats con progresión clara.
4. Mantén tono humano, concreto y fácil de escuchar.
5. Cierra con CTA coherente, sin romper el tono.

## Formato de salida

- `Strategy`
- `Hook Options`
- `Recommended Hook`
- `Segment Outline`
- `Full Script`
- `CTA`

## Enfoque Claude

- Prioriza flow, retención y claridad verbal.
- Si el tema es sensible, suaviza claims absolutos.
- Haz que el guion suene hablado, no redactado como artículo.
