# Examples

## Example Input

```yaml
approved_topic: errores que te dejan sin dinero aunque ganas bien
audience: adultos latinos de 25 a 40 años que trabajan y sienten que no avanzan financieramente
target_duration: 8 minutos
brand_tone: claro, directo, empático y práctico
call_to_action_goal: invitar a seguir una guía inicial de orden financiero
constraints:
  - lenguaje simple
  - no prometer riqueza rápida
  - mantener ejemplos cotidianos
```

## Example Output Summary

### Strategy

- promesa: detectar 7 fugas financieras que sabotean el avance incluso con buen ingreso
- objeción: "yo gano relativamente bien, el problema debe ser que necesito ganar más"
- resultado deseado: que la audiencia vea que puede corregir hábitos concretos desde hoy

### Recommended Hook

"Si ganas mejor que antes pero cada mes terminas igual de ahogado, probablemente no te falta dinero: te están drenando errores que casi nadie nota."

### Segment Outline

1. El problema no siempre es el salario.
2. Error 1 al 3: fugas invisibles.
3. Error 4 al 6: decisiones pequeñas que cuestan muchísimo.
4. Error 7: vivir sin sistema.
5. Cierre con plan simple y CTA.

### CTA

"Si quieres, en el siguiente paso te comparto una guía simple para ordenar tu dinero sin complicarte."

### Voiceover Handoff

- ritmo: conversacional con urgencia suave
- énfasis: frustración inicial y alivio práctico al final
- pausas: después de cada error y antes del CTA
