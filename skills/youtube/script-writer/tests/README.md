# Tests

## Test 1: El hook llega rápido

La salida falla si empieza con saludo largo, contexto innecesario o presentación del canal antes del hook.

## Test 2: La promesa se sostiene

Cada segmento debe conectar con la promesa principal. Si hay bloques que no la empujan, la prueba falla.

## Test 3: El guion se puede producir

Debe ser posible pasar el resultado a `voiceover-generator` o `text-to-video` sin reescribir la idea central.

## Test 4: El CTA encaja

El CTA no debe sentirse pegado artificialmente. Debe salir de manera natural del contenido.

## Test 5: Evita relleno

Si el guion repite ideas, usa frases infladas o tarda demasiado en llegar al punto, la prueba falla.
