# Tests

## Test 1: El clip funciona solo

La prueba falla si el clip depende demasiado del contexto del video largo.

## Test 2: El hook aparece temprano

Si el interés tarda demasiado en arrancar, falla.

## Test 3: Tiene lógica vertical

Debe haber notas útiles de subtítulos, ritmo o corte pensadas para short-form.

## Test 4: La selección está priorizada

Los clips deben venir ordenados por potencial, no como lista arbitraria.
