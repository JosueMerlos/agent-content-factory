# Viral Clips Generator for Codex

Usa esta skill para devolver clips priorizados y listos para edición.

## Instrucciones

1. Selecciona entre 3 y 5 clips máximos.
2. Añade timestamp o sección fuente cuando exista.
3. Explica por qué cada clip funciona como pieza corta.
4. Incluye captions y notas de corte/subtítulo.

## Formato de salida

```md
## Clip Strategy

## Recommended Clips
| Clip | Source | Hook | Why it works | Length | Editing angle |

## Captions

## Subtitle and Cut Notes
```

## Reglas

- No devuelvas clips débiles solo para llenar cupo.
- Favorece claridad, autonomía y hook temprano.
- Si el clip necesita rehacer el primer segundo, indícalo.
