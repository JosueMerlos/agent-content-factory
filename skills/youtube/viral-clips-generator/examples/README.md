# Examples

## Example Input

```yaml
long_video_or_transcript: |
  ...transcripción de un video sobre errores financieros...
shorts_goal: awareness y tráfico hacia el video largo
platform: youtube-shorts
subtitle_style: palabras clave resaltadas en amarillo
constraints:
  - clips de 20 a 35 segundos
  - tono directo
```

## Example Output Summary

### Clip Strategy

- clips cortos con dolor inmediato
- objetivo: generar identificación rápida y curiosidad

### Recommended Clips

1. "Ganas bien y aun así no avanzas"
2. "El error invisible que vacía tu cuenta"
3. "No necesitas ganar más para arreglar esto"

### Captions

- "Si ganas mejor pero sigues igual, mira esto"
- "Este error te quita dinero sin que lo notes"

### Subtitle and Cut Notes

- subtítulos grandes
- resaltar "gano bien" y "no avanzo"
- cortes cada 1 a 2 segundos al inicio
