# Tests

## Test 1: El título tiene promesa clara

La prueba falla si los títulos son vagos o intercambiables.

## Test 2: La descripción aporta valor real

Debe haber una primera línea fuerte y una descripción que no sea puro relleno SEO.

## Test 3: La decisión final está justificada

La salida debe cerrar con `ready_to_publish` o `needs_revision` y una razón clara.

## Test 4: El paquete es utilizable

Debe poder copiarse al flujo de publicación sin rehacerlo casi por completo.
