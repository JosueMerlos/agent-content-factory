# Tests

## Test 1: El output es accionable

Entrada:

- nicho definido
- audiencia definida
- objetivo claro

Debe cumplir:

- prioriza máximo 5 temas,
- elige una idea ganadora,
- entrega handoff usable por `script-writer`.

## Test 2: Evita genericidad

Si la salida contiene ideas vagas como "tips", "consejos" o "mejora tu canal" sin un ángulo concreto, la prueba falla.

## Test 3: La priorización está razonada

Cada tema debe incluir al menos:

- motivo de relevancia,
- puntaje,
- dolor o deseo de audiencia.

## Test 4: Hay continuidad con el pipeline

La recomendación final debe dejar claro:

- qué video se va a producir,
- para quién,
- con qué promesa principal.
